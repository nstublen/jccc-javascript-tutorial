The Math Object

Common mathematical operations can be performed using the JavaScript Math object. 

        var pi = Math.PI;               // PI constant
        Math.floor(pi);                 // Round down (3)
        Math.round(pi);                 // Round to nearest (3)
        Math.ceil(pi);                  // Round up (4)
        
        Math.random();                  // Random between [0, 1)
        
        Math.min(4, 2, 7, -1, 0, -6);   // Minimum value (-6)
        Math.max(4, 2, 7, -1, 0, -6);   // Maximum value (7)

If we want a random integer between 1 and 10, we can use the random and rounding functions. 

        var value = Math.floor((Math.random() * 10) + 1);

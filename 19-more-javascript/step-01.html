for-in Loops

We've seen how to iterate over an array's elements using a for loop. We can also use JavaScript's for-in loop syntax. 

        for (var index = 0; index < myArray.length; ++index) {
            var arrayElement = myArray[index];
            // ...
        }
        
        for (var index in myArray) {
            var arrayElement = myArray[index];
            // ...
        }

Both forms will iterate over the array, but the for-in syntax will avoid iterating elements that are undefined. 

        var myArray = [];
        myArray[4] = "Hello";
        myArray[7] = ", John!";
        
        for (var index in myArray) {
            var arrayElement = myArray[index];
            document.write(arrayElement);
        }
        // Outputs "Hello, John!"

The for-in syntax can also be used to iterate over object properties.

            var myObject = {
                firstName: "John",
                lastName: "Smith"
            };

            for (var propertyName in myObject) {
                var propertyValue = myObject[propertyName];
                document.write(propertyName + " = " + propertyValue + "
");
            }
            // Outputs "firstName = John
lastName = Smith

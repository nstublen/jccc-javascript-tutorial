Split and Join Strings

A couple of common string operations include splitting and joining strings with a specific separator character. JavaScript supports these operations using strings and arrays of strings. 

        var stringList = "1, 2, 3, 4, 5";
        
        var arrayList = stringList.split(", ");
        // ["1", "2", "3", "4", "5"]
        
        var backToString = arrayList.join(";");
        // "1;2;3;4;5"

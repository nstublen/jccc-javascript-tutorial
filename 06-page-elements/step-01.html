Finding Elements

The browser provides a document object that can be used to access elements on the page. 

        // Locates element with id="username"
		document.getElementById("username");

        // Locates all button elements
        document.getElementsByTagName("button");

        // Locates all class="active" elements
        document.getElementsByClassName("active");

Newer browsers will also support CSS-style selectors. 

        // Query the first button with the active class.
        document.querySelector("button.active");

        // Query all buttons with the active class.
        document.querySelectorAll("button.active");

Inserting Elements

Once an object is created it needs to be inserted in the page. 

        // Insert the button as the last child in myDiv.
        myDiv.appendChild(myButton);

        // Remove the button from myDiv.
        myDiv.removeChild(myButton);

        // Insert the myButton before myDiv.
        document.insertBefore(myButton, myDiv);

        // Replace one element with another.
        document.replaceChild(myDiv, myButton);

        var = hasChidren = myDiv.hasChildNodes();

An element will not render on the page until it is inserted within the document.

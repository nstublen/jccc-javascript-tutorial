Variables can be compared with operators common to many other programming languages. 

            a == b;	    // equality
            a != b;     // inequality
            a > b;      // greater than
            a >= b;     // greater than or equal
            a < b;      // less than
            a <= b;     // less than or equal

Comparison statements are evaluated as boolean values - they are true or false. However, JavaScript uses the concept of "truthiness" in its comparisons. The comparison operators will attempt to convert between types if the conversion provides a more meaningful comparison. You can experiment with this conversion in the Chrome Developer Tools console window. 

            "5" == 5;	// true
            "5" > 4;    // true
            8 < "10";   // true
            "8" < "10"; // false

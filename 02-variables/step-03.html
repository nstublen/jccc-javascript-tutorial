Strings

Strings hold text content. They can be defined using single or double quotes. 

            var value1 = "5";
            var value2 = '5.0';

Why would you want to choose one form over the other? 

            var value1 = "What's a string?";
            var value2 = 'He said, "What is a string?"';

Quotation marks can be included by preceding them with a forward slash (\). 

            var value1 = 'What\'s a string?';
            var value2 = "He said, \"What's a string?\"";

Working with Strings

            var text = "Welcome to JavaScript!";
            text.indexOf(" ");      // 7
            text.indexOf(" ", 8);   // 10
            text.lastIndexOf(" ");  // 10
            text.charAt(6);         // "e"
            text[6];                // "e"
            text.substring(8, 10);  // "to"
            text.startsWith("Wel"); // true
            text.endsWith("pt!");   // true
            text.length;            // 22
            text + "!";             // "Welcome to JavaScript!!"

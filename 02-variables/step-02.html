Numbers

Unlike many languages, which distinguish between integers and floating point values, JavaScript numbers are all floating point values. 

            var value1 = 5;
            var value2 = 5.0;

These two variables contain the same value.

Operators

            // +, -, *, /, %
            var number = 6;
            number + 2;

            // ++, --
            number++;
            ++number;

Working with Numbers

            var number = 3.141592;
            number.toString();          // "3.141592"
            number.toFixed(2);          // "3.14"
            number.toPrecision(2);      // "3.1"
            number.toExponential(2);    // "3.14e+0"

Special "Numbers"

            var number = 4 * "car";
            var number = 4 / 0;

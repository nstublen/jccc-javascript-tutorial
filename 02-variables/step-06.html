In addition to true and false, JavaScript also defines undefined and null. 

            var x;               // the value of x is undefined
            x === undefined;     // true
            
            var y = 1;           // the value of y is 1
            y === undefined;     // false

            var z = "undefined"; // the value of z is "undefined"
            z === undefined;     // false
            z === "undefined";   // true

The typeof operator returns a string indicating the type of a variable. 

            var y = "undefined";
            typeof y;               // returns "string"

            y = undefined;
            typeof y;               // returns "undefined"

The null value indicates the variable's value is "nothing", which is distinct from being undefined. 

            var y = "Hello";        // y is assigned a string value
            y = null;               // y is now "nothing"

            y === null;             // true
            y === undefined;        // false

Note that a variable's type is not static. It can changed based on the values assigned to it.

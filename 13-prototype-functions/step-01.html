Sharing Functions

Once we have defined a set of object properties in the constructor, we can write functions designed to operate on those properties. These functions are attached to any object created with our constructor. 

        function ShoppingListItem(itemName, quantity) {
            this.itemName = itemName || "";
            this.quantity = quantity || 1;
            this.complete = false;
        }

        ShoppingListItem.prototype.getQuantity = function () {
            return this.quantity;
        }

        ShoppingListItem.prototype.isComplete = function () {
            return this.complete;
        }

Prototype functions are accessible from any object that was created with the prototype's constructor function. Prototype functions are accessed just like functions that are attached as properties of an object. 

        var item = new ShoppingListItem("socks", 2);
        var quantity = item.getQuantity();
        var complete = item.isComplete();

Inside the prototype function, this references the object that was used to call the function.

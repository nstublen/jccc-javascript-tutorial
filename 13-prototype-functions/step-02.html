Using Prototype Functions

When you reference an object function, JavaScript will first examine the object for a matching function name, then it will search the object's prototype. 

        var item = new ShoppingListItem("socks", 2);

        item.getDoubleQuantity = function () {
            return this.quantity * 2;
        }

        var quantity = item.getQuantity();
        var double = item.getDoubleQuantity();

The getDoubleQuantity function only exists for the single item. If we create another ShoppingListItem, it will not have this function unless we add it. However, all ShoppingListItems will have the prototype functions.

Exercise 7

In this exercise, we will display a popup window in the center of the page and blur the rest of the page content. We will use "blurbox.html" as a starting point and proceed with the following steps:

  1. Add a click event handling function for the "Popup" button.
  2. Connect the event handling function to the "Popup" button.
  3. Add a div that covers the page when the button is clicked.
  4. Place popup content in the center of the new div.

Using new and this

Instead of a "creation" function we would end up with an "initialization" function that looks something like this. 

        function initShoppingListItem(newObj, itemName, quantity) {
            newObj.itemName = itemName || "";
            newObj.quantity = quantity || 1;
            newObj.complete = false;
        }

        var item = {};
        initShoppingListItem(item, "cookies", 24);

Instead of creating an empty object and passing it to a function, we can use JavaScript's new keyword. 

        function ShoppingListItem(itemName, quantity) {
            this.itemName = itemName || "";
            this.quantity = quantity || 1;
            this.complete = false;
        }

        var item = new ShoppingListItem("cookies", 24);

The function doesn't have an argument for the new object, because the new object is referenced using the this keyword.

By convention, constructor functions are capitalized.

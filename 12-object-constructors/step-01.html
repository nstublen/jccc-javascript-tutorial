Initialization Functions

We've seen how we can create a function that will create and initialize an object. 

        function createShoppingListItem(itemName, quantity) {
            return {
                itemName: itemName || "",
                quantity: quantity || 1,
                complete: false
            }
        }

JavaScript provides a convenient way to create objects with such functions. What if we modified our function to receive the object as well as the initial property values? What would that function look like? How would you call that function?

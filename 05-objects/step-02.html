Objects

An object can have properties dynamically added after it has been created. 

		var person = {};            // an empty object
        person.firstName = "John";
		person.lastName = "Smith";
		person.age = 33;

Two object-type variables are equal if they refer to the same object. Unlike numbers, strings, and boolean variables, identical content does not mean equality. 

		var a = {};     // "a" refers to a new object
        var b = {};     // "b" refers to a new object
        
        a === b;        // false: "a" and "b" do not refer to
                        // the same object

        var c = a;      // "c" refers to the same object as "a"
        
        a === c;        // true: "a" and "c" do refer to the
                        // same object
        
        a.value = 6;    // the object referenced by "a" now has
                        // a "value" property
        c.value === 6;  // referencing the object through "c"
                        // references that same property

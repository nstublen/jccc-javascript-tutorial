Arrays

While objects contain a set of property values that are accessed by name, arrays contain a set of values that are accessed by an index number. 

		var list = [1, 2, 3, 4, 5];
        
        list[0] === 1;  // array index values start at zero
        list[1] === 2;
        list[5] === 6;  // false - list[5] is undefined
	
        var list = [];      // create an empty array
        
        list[5] = "John";   // Insert an element in the array
                            // at index position 5; positions
                            // 0-4 are undefined

Array elements do not need to be the same type. 

        var list = [];      // create an empty array
        
        list[0] = "John";   // insert a string
        list[1] = 19;       // insert a number
        list[2] = {};       // insert an empty object
        list[3] = [];       // insert an empty array

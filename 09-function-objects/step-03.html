Functions As Object Properties

Since we can assign functions to variables, we can also assign them to object properties. 

        var calculator = {
            add: function (a, b) {
                return a + b;
            },
            subtract: function(a, b) {
                return a - b;
            }
        };

We can call these functions by referencing them like object properties. 

        var sum = calculator.add(1, 1);
        var difference = calculator.subtract(1, 1);

The document Object

The browser provides an object that represents the current HTML document. We can access elements on the page through functions on the document object. 

        var paragraphs = document.getElementsByTagName("p");
        var element = document.getElementById("p2");
        var classes = document.getElementsByClassName("brush");

These functions return an element object or an array of element objects.

We can also create new elements and insert them into the page. 

        var scrollable =
            document.getElementsByClassName("scrollable");
        scrollable = scrollable[0];
        
        var div = document.createElement("div");
        div.color = "red";
        
        var a = document.createElement("a");
        a.href = "./step-05.html";
        a.innerHTML = "Next Step";
        div.appendChild(a);
        
        scrollable.appendChild(div);

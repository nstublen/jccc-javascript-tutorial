Functions as Function Parameters

Now that we've referenced the function using a variable, what if we passed that variable to another function? 

        function calculateSalesTax(amount) {
            return amount * 0.085;
        }

        function calculateShipping(amount) {
            return amount * 0.085;
        }
        
        var taxCalculator = calculateSalesTax;
        var shippingCalculator = calculateShipping;
        
        var subtotal = 99.95;
        var total = calculateTotal(subtotal, taxCalculator,
                                   shippingCalculator);

We might implement calculateTotal as follows. 

        function calculateTotal(subtotal, taxCalculator,
                                shippingCalculator) {
            var total = subtotal +
                        taxCalculator(subtotal) +
                        shippingCalculator(subtotal);
            return total;
        }

We have called the calculator functions from within another function. However, we could have skipped a step in the first block of code. We could have passed the function names directly to the calculateTotal function: 

        function calculateSalesTax(amount) {
            return amount * 0.085;
        }

        function calculateShipping(amount) {
            return amount * 0.05;
        }
        
        var subtotal = 99.95;
        var total = calculateTotal(subtotal,
                                   calculateSalesTax,
                                   calculateShipping);

First assigning the functions to variables was not necessary. They can be assigned to function parameters by passing them directly to the function.

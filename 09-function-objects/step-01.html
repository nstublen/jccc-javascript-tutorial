Function Variables

JavaScript allows you to assign a function to a variable. 

        function calculateSalesTax(amount) {
            return amount * 0.085;
        }
        
        var calculator = calculateSalesTax; // no parentheses,
                                            // no parameters

You may then call the function through the variable that references it. 

        var tax = calculator(39.95);

The variable calculator refers to the calculateSalesTax function. Think of it as just another name for the function.

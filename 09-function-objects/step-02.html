Functions Without Names

When we declare a variable, we can define its value. 

        var subtotal = 39.95;
        var name = "John Smith";

We can also do this with variables that reference functions. 

        var calculateSalesTax = function (subtotal) {
            return subtotal * 0.085;
        }; // <-- notice semicolon

We have declared a variable, calculateSalesTax and assigned a function to it. This function can be called through the variable name, just like we saw in the last step. The syntax is no different from when we originally declared this function. 

        var tax = calculateSalesTax(subtotal);

Because we declared the function without a name, it is called an anonymous function. (You can specify a name between the function keyword and the parameter list, but you will not be able to reference it from your JavaScript code.

JavaScript is executed as the browser works its way down the page.

You can observe this using the browser's document object to write content to the page.

        <body>
            <script>
                document.write("Hello!");
            </script>
        </body>

You can interleave <script> with HTML to see how the browser executes JavaScript as it renders the page.

This is important to remember since it means that <script> at the top of the page can't access elements that have not yet been rendered by the browser.

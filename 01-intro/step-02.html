Once the page has loaded some JavaScript into the browser, you can access it within <script> tags on the page.

        <head>
            <script src="../js/shCore.js"></script>
            <script src="../js/shBrushJScript.js"></script>

            <link href="../css/shCore.css" />
            <link href="../css/shThemeDefault.css" />
        </head>

You can include the following JavaScript within the <body> element of this page and refresh the page.

        <script type="text/javascript">
            SyntaxHighlighter.all()
        </script>

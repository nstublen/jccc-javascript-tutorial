Below is an example showing how to include JavaScript on a page.

        <body>
            <script>
                // JavaScript goes here...
            </script>
        </body>

You can also load an external JavaScript file, typically in the head section of the page or the end of the body section.

        <head>
            <script src="../js/shCore.js"></script>
            <script src="../css/shBrushJScript.js"></script>

            <link href="../css/shCore.css" />
            <link href="../css/shThemeDefault.css" />
        </head>

You can copy the <script> elements above into the <head> section of this page and refresh the page. This will cause the JavaScript files to be loaded into the browser when the page loads.

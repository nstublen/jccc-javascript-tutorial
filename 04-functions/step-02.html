Unspecified Parameters

Unspecified parameters are undefined when the function executes. 

        function calculateSalesTax(amount) {
            if (amount === undefined) {
                return 0;
            }
            return amount * 0.085;
        }

Assigning default values within a function is one way to implement optional parameters. However, since all parameters are optional, you need to make sure you are calling a function as expected.

Functions

Functions allow you to group statements together into reusable units. 

        function calculateSalesTax(amount) {
            return amount * 0.085;
        }

This function's name is calculateSalesTax. It receives single numeric parameter, amount, and returns a numeric result. 

        var tax = calculateSalesTax(39.95);

It is possible to call a function without passing all the parameters. Try it in the console window and see what happens.

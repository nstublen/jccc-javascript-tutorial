The arguments Property

A better way to determine if parameters were passed to a function is to use the functions arguments property. 

        function calculateSalesTax(amount) {
            if (arguments.length < 1) {
                return 0;
            }
            return amount * 0.085;
        }

The arguments property will allow you to access any extra parameters that were passed to the function. 

            function calculateSalesTax() {
                var subtotal = 0;
                for (var index = 0; index < arguments.length; ++index) {
                    subtotal += arguments[index];
                }
                return subtotal * 0.085;
            }

            var tax = calculateSalesTax(39.95, 41.95, 59.99);

Exercise 4

In this exercise, we will respond to element events and add a shopping checklist item to the list of items on the page. We will use "shoplist.html" as a starting point and proceed with the following steps:

  1. Modify the HTML to include an unordered list that will contain the items in the checklist.
  2. Write a function that will setup event handling for a form's submit button.
  3. When the form is submitted, add the form's data to the shopping checklist.

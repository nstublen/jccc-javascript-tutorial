Accessing Forms

We've seen how to find elements by id, tag name, or class name. In the case of forms, we can also use a special document property. 

        var firstForm = document.forms[0];
        var formById = document.forms["my-form-id"];

The forms property contains an array of all the forms on the page. Each form can be accessed using its index position in the array, or the id or name attribute specified in the HTML.

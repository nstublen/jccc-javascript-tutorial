Accessing Form Inputs

Each form object provides access to its inputs through an elements property. 

        var firstForm = document.forms[0];
        var inputs = firstForm.elements;

We can iterate the inputs using a for loop. 

        for (var index = 0; index < inputs.length; ++index) {
            var name = inputs[index].name;
            var value = inputs[index].value;
            values.innerHTML += name + " = " + value + "
";
        }

Objects

A JavaScript object is a flexible data type that stores multiple values. Each value is identified by a unique property name within the object. The syntax for declaring an object is similar to CSS. 

		var person = {
		    firstName: "John",
			lastName: "Smith",
			age: 33
		};

An object property can be another object. 

		var person = {
		    firstName: "John",
			lastName: "Smith",
			age: 33,
			address: {
			    street: "12345 College Blvd",
                city: "Overland Park",
                state: "KS",
                zip: "66210"
			}
		};

Object properties are accessed using a dot or bracket notation. 

		person.firstName;
        person['firstName'];

		person.lastName;
        person['lastName'];

		person.address.state;
		person.address['state'];
        person['address'].state;
        person['address']['state'];

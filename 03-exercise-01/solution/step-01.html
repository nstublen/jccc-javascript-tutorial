Exercise 01 - Variables

Exercise 1-A

In this exercise, we will parse a string representing a date to determine the month, day, and year.

We will use the files in the "03-exercise-01" folder as a starting point and proceed with the following steps:

  1. Declare a string variable containing a date in the format mm-dd-yyyy.
  2. Find the index position of the two dashes.
  3. Extract the month, day, and year into separate string variables.
  4. Handle the date in the format yyyy-mm-dd as well as mm-dd-yyyy.
  5. Determine if the date is the first day of the year.

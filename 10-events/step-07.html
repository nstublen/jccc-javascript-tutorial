Event Objects

Events will pass an object to the event handler that describes the event. 

        element.onclick = function (e) {
        };

The properties of the event will vary based on the type of the event, but there are some common properties and functions. 

        element.onclick = function (e) {
            e.target;   // element that triggered the event
            e.type;     // name of the event

            // cancel default action
            e.preventDefault();

            // prevent further event callbacks
            e.stopImmediatePropagation();

            // prevent further propagation
            e.stopPropagation();
        };

Handling Events

Interacting with the elements on a page will generate events. Events can be accessed as properties on an element. 

        // Assign a function to the element property.

        function myClickFunction() {
            alert("You clicked it!");
        }

		element.onclick = myClickFunction;
    
        // Or use an anonymous function.

        element.onclick = function () {
            alert("You clicked it!");
        };

Events can also be connected using the element's HTML attributes. 

        
Hello!

In this case, the attribute value contains JavaScript that will be executed; it's not simply the name of the function.
